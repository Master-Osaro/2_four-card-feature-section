### What I learned 

```css media queries
@media only screen and (max-width: 992px) {

    .features section{
        max-width: 400px;
        padding: 16px;
        margin: 16px;
    }
}

@media only screen and (max-width: 690px) {
    .features{
        margin: auto;
        display: flex;
        flex-direction: column;
        flex: 1;
        justify-content: center;
        align-items: center;
    }
}
```
